An alternative to the JavaScript _eclint_ written in Go.

## Usage

```
$ go install gitlab.com/greut/eclint

$ eclint -version
```

## Features

- `charset`
- `end_of_line`
- `indent_style`
- `trim_trailing_whitespace`

## Missing features

- `indent_size`
- `insert_final_newline`
- more tests
- ignoring `.git`, `vendor`, etc.
- etc.
