# eclint - EditorConfig linter

An alternative to the JavaScript _eclint_ written in Go.

## Usage

```
$ go install github.com/greut/eclint

$ eclint -version
```

## Features

- `end_of_line`
- `indent_style`

## Missing features

- `charset`
- `indent_style`
- tests
- etc.
